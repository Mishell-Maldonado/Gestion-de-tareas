Gestión de Tareas - Sistema con TAD (Pilas y Colas)
Este proyecto es una aplicación de consola desarrollada en Java para la gestión eficiente de actividades, aplicando conceptos de Tipos Abstractos de Datos (TAD) dinámicos estudiados en la asignatura de Algoritmos y Estructuras de Datos.

 Funcionalidades
El sistema permite administrar el flujo de trabajo mediante dos estructuras principales:

Pila (Stack): Gestiona las tareas pendientes bajo el principio LIFO (Last-In, First-Out). La última tarea agregada es la próxima a realizar.

Cola (Queue): Almacena el historial de tareas completadas siguiendo el principio FIFO (First-In, First-Out).

 Tecnologías Utilizadas
Lenguaje: Java SE

Estructuras: java.util.Stack y java.util.LinkedList (Queue)

Herramientas: Visual Studio Code y Git para el control de versiones.

Instrucciones de Uso

1.Ejecutar la clase principal gestiondetarea.java.

2.Utilizar el menú interactivo para agregar, completar o eliminar tareas.

3.Consultar la "Próxima tarea" para visualizar la cima de la pila sin extraer el dato.
